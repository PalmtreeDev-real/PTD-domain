What is this?

This is an example site running on a local .ptd (PalmtreeDev Domain) top-level domain. Every *.ptd domain resolves to 127.0.0.1 on this machine, served by dnsmasq + nginx.

Getting Started

Create your own .ptd projects by adding directories to /var/www/ptd/ and running:

sudo /usr/local/bin/ptd add <project-name>

Each project is immediately available at http://<project-name>.ptd/
